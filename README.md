# Industry-Academia-Platform (Capstone-Project)
Developed as a team of 3, a role-based web platform connecting students, companies, and universities for real-world project collaboration. Built with React, Node.js/Express, and MySQL, featuring JWT authentication and RESTful API architecture. RBAC across four user panels, a gamified XP/scoring system, and academic reporting modules are being implemented following Agile/Scrum methodology. The project is ongoing — planned next steps include completing full backend API integration, connecting the frontend to live endpoints, and implementing real-time in-platform messaging and notification modules.
